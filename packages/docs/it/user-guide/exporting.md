# Esportazione

## Esportazione immagini
- **Scala** — 0,5× a 4× · **Formato** — PNG, JPG, WEBP
- **⇧ ⌘ E** o click destro → Esporta… o pulsante nel pannello proprietà

## Operazioni file .fig
| Azione | Mac | Windows / Linux |
|--------|-----|-----------------|
| Apri | ⌘ O | Ctrl + O |
| Salva | ⌘ S | Ctrl + S |
| Salva come | ⇧ ⌘ S | Shift + Ctrl + S |

Compatibilità round-trip con Figma.
