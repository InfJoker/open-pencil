# Fonctionnalités

## Pourquoi OpenPencil

Alternative open-source aux outils de design : MIT, compatible Figma, entièrement local et programmable.

## Import & export .fig
Ouvre et enregistre les fichiers Figma natifs. Schema Kiwi complet (194 définitions). Round-trip fidèle.

## Copier-coller avec Figma
Copier dans Figma, coller dans OpenPencil et vice versa. Format binaire Kiwi identique.

## Outils de forme
Rectangle (R), Ellipse (O), Ligne (L), Polygone, Étoile. Remplissage, contour, effets.

## Mise en page auto
Yoga WASM (flexbox). Direction, gap, padding, alignement. ⇧A pour activer.

## Édition de texte
Édition native sur le canevas. Sélecteur de polices virtuel. Formatage riche (B/I/U/S).

## Annuler/Rétablir
⌘Z / ⇧⌘Z. Pattern inverse-command. Toute opération est annulable.

## Composants et instances
Composants réutilisables (⌥⌘K), jeux (⇧⌘K), sync live, surcharges.

## Variables
Jetons de design avec collections et modes. Liaison aux remplissages.

## Chat IA
29 outils via OpenRouter. Serveur MCP (stdio + HTTP).

## CLI
info, tree, find, export, analyze, eval. Toutes les commandes supportent `--json`.
