# Exportation

## Export d'images
- **Échelle** — 0,5× à 4× · **Format** — PNG, JPG, WEBP
- **⇧ ⌘ E** ou clic droit → Exporter… ou bouton dans le panneau propriétés

## Opérations fichiers .fig
| Action | Mac | Windows / Linux |
|--------|-----|-----------------|
| Ouvrir | ⌘ O | Ctrl + O |
| Enregistrer | ⌘ S | Ctrl + S |
| Enregistrer sous | ⇧ ⌘ S | Shift + Ctrl + S |

Compatibilité aller-retour avec Figma.
