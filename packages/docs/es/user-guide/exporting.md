# Exportar

## Exportación de imágenes
- **Escala** — 0,5× a 4× · **Formato** — PNG, JPG, WEBP
- **⇧ ⌘ E** o clic derecho → Exportar… o botón en panel de propiedades

## Operaciones de archivo .fig
| Acción | Mac | Windows / Linux |
|--------|-----|-----------------|
| Abrir | ⌘ O | Ctrl + O |
| Guardar | ⌘ S | Ctrl + S |
| Guardar como | ⇧ ⌘ S | Shift + Ctrl + S |

Compatibilidad de ida y vuelta con Figma.
