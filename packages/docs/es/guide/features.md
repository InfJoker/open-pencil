# Características

## Por qué OpenPencil

Alternativa open-source: MIT, compatible con Figma, completamente local y programable.

## Import & export .fig
Abre y guarda archivos Figma nativos. Schema Kiwi completo (194 definiciones). Round-trip fiel.

## Copiar y pegar con Figma
Copiar en Figma, pegar en OpenPencil y viceversa. Mismo formato binario Kiwi.

## Herramientas de forma
Rectángulo (R), Elipse (O), Línea (L), Polígono, Estrella. Relleno, trazo, efectos.

## Auto-layout
Yoga WASM (flexbox). Dirección, gap, padding, alineación. ⇧A para activar.

## Edición de texto
Edición nativa en el lienzo. Selector de fuentes virtual. Formato enriquecido (B/I/U/S).

## Deshacer/Rehacer
⌘Z / ⇧⌘Z. Patrón inverse-command. Toda operación es reversible.

## Componentes e instancias
Componentes reutilizables (⌥⌘K), conjuntos (⇧⌘K), sincronización live, overrides.

## Variables
Tokens de diseño con colecciones y modos. Vinculación a rellenos.

## Chat IA
29 herramientas vía OpenRouter. Servidor MCP (stdio + HTTP).

## CLI
info, tree, find, export, analyze, eval. Todos los comandos soportan `--json`.
