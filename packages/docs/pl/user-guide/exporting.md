# Eksportowanie

## Eksport obrazów
- **Skala** — 0,5× do 4× · **Format** — PNG, JPG, WEBP
- **⇧ ⌘ E** lub prawy przycisk → Eksportuj… lub przycisk w panelu właściwości

## Operacje na plikach .fig
| Akcja | Mac | Windows / Linux |
|-------|-----|-----------------|
| Otwórz | ⌘ O | Ctrl + O |
| Zapisz | ⌘ S | Ctrl + S |
| Zapisz jako | ⇧ ⌘ S | Shift + Ctrl + S |

Kompatybilność round-trip z Figmą.
